
# 💎 Strike Rich

A C# Windows Forms game where a poor farmer searches a chain of islands for a hidden diamond. Every island is a gamble—you might strike it rich, find nothing, or fall into the water!

---

## 🎮 Gameplay

- Press **Start** to begin.
- Press **Go** to move to the next island.
- Click an island to dig.
- 💎 Find the **diamond** to win.
- 🌊 Hit **water** and the game is over.
- 🕳️ Find an **empty hole** and continue searching.

---

## ✨ Features

- Randomized island outcomes
- Win and lose conditions
- Interactive gameplay
- Simple and clean UI
- Built with C# WinForms
